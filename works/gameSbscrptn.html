"NEO" ゲーミングサブスクサイト

Web Design / html / css / JavaScript

2022 - 【学校課題】"NEO" ゲーミングサブスク ウェブサイト

専門学校の授業課題で制作したウェブサイト。現在はドメイン、サーバーともに存在しないが、制作内容を紹介。

Production Process

専門学校1年生の頃の授業課題。自分の「あったらいいなと思うサブスクサービス」というテーマで制作した。ロゴやアイコンなど含めデバイスの画像以外はすべて自作。
ゲーミングデバイス(マウスやキーボード、オーディオ類など)の良さは触ってからじゃないとわからない部分が多くあるため、2万円近く支払って購入するしかないという悩みを解決させる案として、「サブスクサービスで様々なデバイスを使い放題にして、手に取ってみた感触などで購入する商品を決める」というサービスを提案した。

サイバー感のあるデザイン

蛍光緑のような色をキーカラーに選出。サイバー感のあるゲーミングデバイスらしいメインビジュアルに仕上がっている。

ロゴ

注文した商品を素早く配達するイメージと、スタイリッシュさを表現するために三角形の形がところどころに入っている。また、「NEOを通してより良いゲーミングライフを送ってほしい」という思いを右向きの三角形で表している。

トップページ

NEOのサービス紹介やメリットを記載。ピックアップアイテムなども分かりやすい位置に。

商品一覧ページ

シンプルだが、サイトのイメージに合わせた配色や配置にこだわりを。

商品ページ

商品のメインビジュアルを活かす形で、機構なども確認できるようなレイアウト。
