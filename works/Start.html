Start - ワタリドリ / [Alexandros]

Anime MAD / Text Animation / Motion Graphics / Graphic Design / 3D Camera

2022.3.1 -【複合MAD】「Start」ワタリドリ / [Alexandros]

テーマは「旅立ち」。この作品自体が、自身の「MAD」動画作品の処女作。
「明日への希望を見た人に届けたい」という思いが込められている。
YouTubeの再生回数は30万回を超え、たくさんの人に支持されている作品だ。

Production Process

制作時期は高校三年生の3学期。卒業と旅立ちを目の前にした自分が「卒業作品」としてYouTubeに投稿した。
自身の最も好きなアーティスト「[Alexandros]」による楽曲「ワタリドリ」を採用し、当時の自分の内に秘めていた"表現したいもの"を存分に発揮した作品だ。

サムネイル

サムネイルはMADらしさを全開にしたビジュアル。様々な作品のキャラクターによる強い眼光が「旅立ち」への「決意」や「希望」をあらわしているようにも見える。

世界観とフォント

採用している作品や、シーン、そして歌詞との調和を大切にして構成されたシーンが多数存在している。フォントや文字カラー、配置に注目して観ると楽しめるだろう。
