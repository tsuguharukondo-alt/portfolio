Portfolio Site

Web Design / html / css / JavaScript

2026.6.5 - Sheepe Fujitsugu aka Tsuguharu Kondo Portfolio Site

自身の作品をまとめたポートフォリオサイト。サイト自体も自分の作品であり、多種多様なスキルを表現している。
※スナップショットは、一部本来の見え方と違う場所がございます。

Production Process

デザインとある程度のコーディングは専門学校時代に制作した。月日がたち、再就職のタイミングでコーディングとサイト設計の見直しをし、リリース。

元々のサイト

専門学校1年の頃に授業でポートフォリオサイトを制作したのが、このサイトの原型だ。
当時はWebデザインのいろはも無かったため、学生が作ったシンプルなサイトだった。インパクトやブランディング力に欠ける部分が多く見られた。また、作品を一覧でしか見せず、詳しい魅力などが伝わりづらい構成だった。

メインビジュアル

大まかな構成はあまり変わっていない。自身のREELが背景で流れているメインビジュアル。ただ、ヘッダーの仕様や、テキストの内容などを一新。クリエイターらしい洗練されたデザインにアップグレードした。

イントロダクション

「ポートフォリオサイトを閲覧した時、メインビジュアルを見た後に気になることはなんだ」と考えた。個人的には「一体どんなクリエイターなんだろう」ということだと考えた。「動画」「グラフィック」「WEB」大まかに3つの武器を携えている自分だからこそ、その多様さ、器用さをアピールすることにした。
RGBの三色でカテゴリーを分け、自分のスキルと使用経験のあるツールを列挙し、「手始めに自己紹介を」といった心意気が感じられるデザインに落とし込んだ。

それぞれのカテゴリー

イントロダクション後はそれぞれのカテゴリーのピックアップ作品を掲示することでした。背景色で何のカテゴリーなのかを表している。

リブランディング

Aboutページでは、世界観づくりのためのセクションを用意し、おしゃれさとカッコよさを追及している。

プロフィール

プロフィール欄も、旧デザインから一新。経歴なども書き込める場所を作り、より解像度を高めるセクションに仕上げている。

作品一覧

作品一覧も一新し、タイトルと何の作品なのかを簡略的に伝えている。

各作品ページ

各作品ページでは、メインビジュアルにサムネイルを大きく見せ、トップページの構造に似た造りをしている。
タイトルや簡易的な説明をはじめで行う。

スナップショット

スクロールしていくと、見どころをスナップショットで掲載。説明書きも追加することでより詳しく作品に触れることができる。

クレジット

最後にクレジットを載せることで、だれが携わり、どこの何を担当したのかなどの情報を確認できる。
アニメーションや、サイトの質感はこの後是非、自身の手で確かめてほしい。
